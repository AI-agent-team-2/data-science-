# Назначение

Насосные группы быстрого монтажа ONDO представляют собой готовые модули для упрощения монтажа систем отопления с несколькими независимыми контурами.

Каждый контур системы оснащается собственным циркуляционным насосом, что обеспечивает стабильную и эффективную работу всей системы отопления.

Насосные группы с трёхходовым смесителем применяются в контурах, где требуется регулирование температуры подачи, например:

- радиаторное отопление
- системы теплого пола
- вентиляционные контуры

В качестве теплоносителя допускается использование:

- воды
- растворов гликоля

Максимальное содержание гликоля: до 40%.

# Монтаж

Установка насосных групп допускается только в отапливаемых помещениях.

Перед монтажом необходимо очистить трубопровод от загрязнений:

- окалина
- ржавчина
- строительный мусор

Система отопления должна быть промыта до визуальной чистоты воды.

После монтажа требуется выполнить испытание герметичности системы методом манометрических испытаний.

# Варианты установки

Насосные группы могут устанавливаться:

- непосредственно в систему отопления
- на распределительный коллектор

При использовании гидравлического разделителя схема подключения должна соответствовать документации разделителя.

# Перестановка подачи и обратки

A — запорный узел на подающей линии (термометр с красным кольцом)

B — запорный узел на обратной линии с обратным клапаном (термометр с синим кольцом)

Для изменения направления необходимо:

- поменять местами левую и правую стороны группы
- повернуть циркуляционный насос на 180°
- повернуть электрическую часть насоса на 180°

# Регулировка трёхходового клапана

Для перестановки смесительного клапана необходимо:

1. открутить винты фиксации ручки клапана
2. снять заднюю заглушку
3. перевернуть блок клапана
4. установить ручку и заглушку обратно
5. установить указатель ручки на шкале VL links или VL rechts
6. установить ручку под углом 45°
7. собрать привод

# Регулировка байпаса

Трёхходовой клапан оснащён байпасом для снижения температурных колебаний в регулируемом контуре.

Регулировка выполняется поворотом шлица байпаса.

Диапазон регулировки: 0°–90°.

Максимальное открытие достигается при совпадении шлица с меткой на клапане.

# Сервопривод

Насосная группа может оснащаться трёхпозиционным сервоприводом для управления через систему автоматики.

Установка выполняется согласно инструкции производителя.

# Обратный клапан

В шаровом кране обратной линии установлен обратный клапан.

Он предотвращает естественную циркуляцию теплоносителя при выключенном насосе.

Для отключения клапана при заполнении системы необходимо повернуть ручку на 45° по часовой стрелке.

# Эксплуатация

Оборудование должно эксплуатироваться при параметрах давления и температуры, указанных в технических характеристиках.

Не допускается:

- механическое повреждение корпуса
- короткие замыкания
- загрязнение оборудования

# Хранение и транспортировка

Хранение осуществляется в упаковке производителя.

Условия хранения: категория 3 по ГОСТ 15150-69.

Транспортировка: категория 5 по ГОСТ 15150-69.
