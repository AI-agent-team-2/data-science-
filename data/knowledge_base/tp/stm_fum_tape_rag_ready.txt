# Назначение

ФУМ‑лента применяется для герметизации резьбовых соединений трубопроводов.

Используется в трубопроводах, транспортирующих:

- воду
- жидкости
- газы

Рабочие среды должны быть неагрессивными к материалу ленты — политетрафторэтилену (PTFE, фторопласту).

# Материал

Лента изготовлена из неспеченной пленки политетрафторэтилена (PTFE, фторопласт‑4).

Производство осуществляется методом прокатки жгута.

Лента намотана на полипропиленовую бобину и имеет защитную крышку.

# Применение

Перед намоткой ФУМ‑ленты необходимо очистить резьбу соединяемых деталей от:

- грязи
- ржавчины
- посторонних частиц

Лента наматывается по ходу резьбы.

Каждый следующий виток должен перекрывать предыдущий примерно на:

30–40 %.

Намотка должна выполняться с небольшим натяжением для предотвращения:

- морщин
- складок

После намотки ленту необходимо прижать к резьбе пальцами.

Если соединяемая деталь накручивается слишком легко, необходимо добавить дополнительное количество витков ленты.

После выполнения соединения оно может сразу эксплуатироваться под рабочей нагрузкой.

Повторное использование ранее примененной ленты не допускается.
