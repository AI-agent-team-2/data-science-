
DOCUMENT: Смесители пристенные ROEGEN

PRODUCT: Пристенные смесители для ванной, душа и кухни
BRAND: ROEGEN

# Назначение

Смесители ROEGEN предназначены для:

- смешивания горячей и холодной воды
- регулирования температуры воды
- регулирования расхода воды
- подачи и остановки потока воды

В ассортимент входят смесители для:

- кухни
- раковины
- ванны
- душа

Также выпускаются моно-краны для горячей или холодной воды.

Назначение смесителя определяется первой цифрой артикула.

# Типы смесителей по артикулу

Rx0**x — смесители для кухни и раковины  
Rx1**x — смесители для ванны через излив  
Rx2**x — смесители для ванны и душа через лейку  
Rx4**x — смесители для душа  
Rx5**x — моно-краны

# Технические характеристики

Материал корпуса: металлический сплав

Минимальное давление воды: 0.1 МПа

Рабочее давление: 0.1 – 0.6 МПа

Максимальное давление: 1 МПа

Максимальная рабочая температура горячей воды: 60 °C

Максимально допустимая температура горячей воды: 80 °C

Расход воды при давлении 0.3 МПа:

- смесители для кухни — не менее 0.2 л/с
- смесители для ванны — не менее 0.33 л/с
- смесители для душа — не менее 0.2 л/с

Если давление в системе превышает 0.6 МПа, рекомендуется установка редуктора давления.

# Тип крепления

Пристенные смесители устанавливаются непосредственно на выводы труб водоснабжения из стены.

Подключение выполняется через:

- эксцентрики
- прокладки
- декоративные отражатели

Резьба подключения: 1/2".

# Комплект поставки

Пристенные смесители поставляются в комплекте:

- смеситель
- эксцентрики
- прокладки
- декоративные отражатели

Смесители для ванны дополнительно комплектуются:

- изливом
- дивертором
- душевым шлангом
- ручной душевой лейкой
- держателем для душевой лейки

# Монтаж

Основные этапы установки:

1. Установить эксцентрики в выводы трубопровода.
2. Установить декоративные отражатели.
3. Подключить корпус смесителя к эксцентрикам.
4. Затянуть соединения с использованием мягких прокладок.

Допустимое отклонение осей трубопроводов не должно превышать 10 градусов.

Перед установкой рекомендуется промыть систему водоснабжения.

# Эксплуатация

Запрещается использовать смеситель в условиях замерзания воды.

Для корректной работы рекомендуется:

- использовать фильтры очистки воды (не менее 100 мкм)
- устанавливать редукторы давления при большой разнице давления в контурах

# Уход за смесителем

Запрещается использовать для очистки:

- абразивные порошки
- кислоты
- ацетон
- щелочные растворы
- хлорсодержащие средства

Рекомендуется применять мягкие моющие средства.

# Гарантия

Гарантийный срок эксплуатации: 36 месяцев.

При установке в общественных местах гарантия составляет 6 месяцев.

Полная гарантия распространяется на литой корпус смесителя.

Ограниченная гарантия 2 месяца действует на следующие комплектующие:

- картриджи
- диверторы
- изливы
- задние гайки
- ручки и маховики

# Ограничения гарантии

Гарантия не распространяется на:

- шланги
- лейки
- прокладки
- аэраторы
- гибкую подводку

Также гарантия аннулируется при:

- отсутствии фильтров очистки воды
- отсутствии редуктора давления при значительной разнице давлений
- неквалифицированной установке
- использовании неоригинальных комплектующих
- механических повреждениях
- наличии известкового налета

# Срок службы

При правильной установке и эксплуатации срок службы смесителя может превышать 5 лет.
