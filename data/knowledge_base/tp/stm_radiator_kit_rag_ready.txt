
DOCUMENT: Комплекты для радиаторов СТМ ТЕРМО

BRAND: СТМ ТЕРМО
PRODUCT: Комплекты монтажные для алюминиевых и биметаллических радиаторов

SOURCE: технический паспорт изделия

# Назначение

Комплекты для радиаторов предназначены для монтажа и подключения
алюминиевых и биметаллических радиаторов отопления.

Комплекты применяются в системах:

- водяного отопления
- системах с теплоносителем вода
- системах с низкозамерзающими жидкостями

Радиаторы должны иметь присоединительный размер коллекторов:

G1” (внутренняя резьба).

Переходники и заглушки выпускаются
с правой (D) и левой (S) резьбой.

# Артикулы комплектов

CARMKB12 — комплект для радиатора 1/2" с кронштейнами  
CARMKB34 — комплект для радиатора 3/4" с кронштейнами  

CARMK012 — комплект для радиатора 1/2" без кронштейнов  
CARMK034 — комплект для радиатора 3/4" без кронштейнов  

CARMKT12 — комплект для радиатора 1/2" с тремя кронштейнами  
CARMKT34 — комплект для радиатора 3/4" с тремя кронштейнами

# Состав комплектов

Типовой комплект содержит:

- переходники для радиатора левый и правый
- ручной воздухоотводчик (кран Маевского)
- заглушку
- ключ для воздухоотводчика
- кронштейны (в зависимости от комплектации)

# Переходники для радиаторов

CARPLR12 — переходник 1" × 1/2" левый  
CARPRR12 — переходник 1" × 1/2" правый  

CARPLR34 — переходник 1" × 3/4" левый  
CARPRR34 — переходник 1" × 3/4" правый

Назначение:

используются для присоединения трубопроводов
и арматуры к радиаторам с коллектором 1".

Основные характеристики:

максимальное давление:
1.6 МПа

уплотнение:
силиконовая прокладка

материал:
никелированная латунь

момент затяжки:
не более 25 Нм

# Воздухоотводчик (кран Маевского)

CARDP012 — 1/2"  
CARDP034 — 3/4"

Назначение:

ручной выпуск воздуха и газов из радиаторов.

Характеристики:

максимальная температура:
110 °C

рабочее давление:
1.0 МПа

материалы:

корпус — латунь SW 617N  
винт — латунь SW 614N  
оголовок — нейлон  
уплотнение — EPDM

Управление:

- специальным ключом
- шлицевой отверткой

# Заглушка радиатора

CARPA012 — 1/2"  
CARPA034 — 3/4"

Материал:

никелированная латунь CW 614N
с покрытием эпоксидной эмалью RAL 9016

максимальное давление:
1.0 МПа

уплотнение:
EPDM O‑ring

# Ключ для воздухоотводчика

CARKPLDP

Материал:

нейлон

Размер под ключ:

квадрат 5×5 мм

# Кронштейн для радиатора

CARHSD01

Назначение:

настенная установка алюминиевых
и биметаллических радиаторов.

Материалы:

кронштейн — оцинкованная углеродистая сталь  
дюбель — полипропилен

характеристики:

максимальная нагрузка:
350 Н

вылет до посадки:
55 мм

# Эксплуатация

При выпуске воздуха из радиаторов
запрещается использовать открытый огонь,
так как в выделяющемся газе может присутствовать водород.

# Хранение

Хранение осуществляется по условиям 3 ГОСТ 15150.

# Транспортировка

Транспортировка осуществляется по условиям 5 ГОСТ 15150.

# Гарантия

Гарантийный срок:

24 месяца.
