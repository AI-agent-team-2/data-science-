
DOCUMENT: Труба для теплого пола PEX‑EVOH СТМ ПЛАСТ

BRAND: СТМ ПЛАСТ
PRODUCT: Труба из сшитого полиэтилена PEX‑b с кислородным барьером EVOH

# Назначение

Труба PEX‑EVOH применяется в системах:

- питьевого и хозяйственно‑питьевого водоснабжения
- горячего водоснабжения
- водяного отопления
- систем водяного теплого пола
- теплых стен
- почвенного подогрева
- технологических трубопроводов для жидкостей, неагрессивных к материалу трубы

Рабочий слой трубы изготовлен из сшитого полиэтилена PEX‑b.

Наружный слой выполнен из EVOH — кислородного барьера, предотвращающего диффузию кислорода в теплоноситель.

Наружный и внутренний слои соединены через эластичный клеевой слой.

# Монтаж

Монтаж труб должен выполняться при температуре окружающей среды не ниже 10 °C.

Для соединения рекомендуется использовать:

- пресс‑фитинги
- обжимные фитинги

Во время монтажа не допускаются:

- сплющивания трубы
- переломы трубопровода

При повреждении трубы участок должен быть удален.

Бухты труб, хранившиеся при температуре ниже 0 °C, должны выдерживаться 24 часа при температуре не ниже 10 °C.

При изгибе трубы с радиусом близким к предельному (5D) рекомендуется прогрев трубы строительным феном до 130 °C.

Для изгиба рекомендуется использовать пружинный кондуктор.

Чтобы избежать распрямления трубы (эффект памяти), в местах изгиба необходимо фиксировать трубу хомутами с шагом 10 см.

Механическое повреждение слоя EVOH увеличивает кислородопроницаемость трубопровода.

Трубу необходимо защищать от воздействия прямых солнечных лучей.

# Ограничения эксплуатации

Трубы PEX‑EVOH нельзя применять:

- при температуре транспортируемой жидкости выше 90 °C
- при давлении выше указанного в технических характеристиках
- в помещениях категории «Г» по пожарной безопасности
- в помещениях с источниками теплового излучения выше 150 °C
- в системах центрального отопления с элеваторными узлами
- для расширительных и сигнальных трубопроводов

# Хранение

Трубы допускается перевозить любым видом транспорта.

Бухты труб должны перевозиться в крытом транспорте.

Трубы необходимо укладывать на ровную поверхность без острых предметов.

Высота штабеля при хранении не должна превышать 3 м.

Хранение выполняется по условиям 5 ГОСТ 15150.

Трубы должны быть защищены от прямых солнечных лучей.
