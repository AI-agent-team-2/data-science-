
DOCUMENT: Комплектующие для полотенцесушителей СТМ

BRAND: СТМ
PRODUCT: Комплектующие для подключения полотенцесушителей

SOURCE: технический паспорт изделия

# Назначение

Комплектующие для полотенцесушителей предназначены для
подключения полотенцесушителей к системам:

- горячего водоснабжения
- водяного отопления

# Материал

Латунь с хромированным покрытием

# Основные характеристики

Рабочая температура:
до 100 °C

Рабочее давление:
до 10 бар

# Монтаж

Комплектующие устанавливаются путем вкручивания
в трубопровод с использованием резьбовых уплотнителей
(например ФУМ‑ленты).

При монтаже необходимо:

- использовать подходящий инструмент
- не применять чрезмерное усилие
- обеспечить герметичность соединений

# Хранение

Хранение осуществляется по условиям 3 ГОСТ 15150.

# Транспортировка

Транспортировка осуществляется по условиям 5 ГОСТ 15150.

# Гарантия

Гарантийный срок:
24 месяца.
