
DOCUMENT: Смесители настольные ROEGEN

PRODUCT: Смесители для кухни, раковины и ванны
BRAND: ROEGEN

# Назначение

Смесители ROEGEN предназначены для:

- смешивания горячей и холодной воды
- регулирования температуры воды
- регулирования расхода воды
- пуска и остановки потока воды

Также в ассортимент входят моно‑краны для горячей или холодной воды.

Смесители применяются для:

- кухни
- раковины
- ванны
- душа

Назначение модели определяется первой цифрой артикула.

# Типы смесителей

Rx0**x — смесители для кухни и раковины  
Rx1**x — смесители для ванны через излив  
Rx2**x — смесители для ванны и душа через лейку  
Rx4**x — смесители для душа  
Rx5**x — моно‑краны для горячей или холодной воды

# Технические характеристики

Материал корпуса: металлический сплав

Минимальное давление воды: 0.1 МПа

Рабочее давление: 0.1 – 0.6 МПа

Максимальное давление: 1 МПа

Максимальная рабочая температура горячей воды: 60 °C

Максимально допустимая температура горячей воды: 80 °C

Расход воды при давлении 0.3 МПа:

- смесители для кухни — не менее 0.2 л/с
- смесители для ванны — не менее 0.33 л/с
- смесители для душа — не менее 0.2 л/с

При давлении в системе более 0.6 МПа рекомендуется установка редуктора давления.

# Типы крепления

Смесители ROEGEN делятся на:

- настольные
- пристенные

Пристенные смесители:

- крепятся к выходам трубопровода в стене
- используют эксцентрики с резьбой 1/2"

Настольные смесители:

- устанавливаются через отверстие в мойке или раковине
- подключаются гибкой подводкой 1/2"

# Комплект поставки

Смесители поставляются в комплекте, достаточном для установки.

Настольные смесители включают:

- крепежный набор
- прижимную гайку или шпильки

Пристенные смесители включают:

- эксцентрики
- прокладки
- декоративные отражатели

Смесители для ванны дополнительно комплектуются:

- изливом
- дивертором
- душевым шлангом
- душевой лейкой
- кронштейном для лейки

# Монтаж

Смеситель устанавливается через отверстие в мойке или раковине.

Основные этапы установки:

1. Установить смеситель в монтажное отверстие.
2. Установить прокладку.
3. Закрепить смеситель прижимной гайкой или шпильками.
4. Подключить гибкую подводку к водопроводу.

Перед установкой рекомендуется промыть систему водоснабжения.

# Эксплуатация

Не допускается использование смесителя в условиях замерзания воды.

При эксплуатации рекомендуется:

- использовать фильтры очистки воды
- устанавливать редуктор давления при высоком давлении
- регулярно очищать аэратор

# Уход за смесителем

Запрещается использовать для очистки:

- абразивные порошки
- кислоты
- ацетон
- щелочные растворы
- хлорсодержащие вещества

Рекомендуется использовать мягкие моющие средства.

# Гарантия

Гарантийный срок эксплуатации: 36 месяцев.

При установке в общественных местах гарантия составляет 6 месяцев.

На корпус смесителя распространяется полный гарантийный срок.

На отдельные комплектующие действует ограниченная гарантия 2 месяца:

- картриджи
- диверторы
- изливы
- гайки
- ручки

Расходные материалы не подлежат замене по гарантии:

- шланги
- лейки
- прокладки
- аэраторы

# Условия отказа в гарантийном ремонте

Гарантия не распространяется при:

- отсутствии фильтров очистки воды
- отсутствии редуктора давления при значительной разнице давлений
- неквалифицированной установке
- использовании неоригинальных запчастей
- механических повреждениях
- наличии известкового налета
- использовании агрессивных чистящих средств

# Срок службы

При правильной эксплуатации срок службы смесителя может превышать 5 лет.
