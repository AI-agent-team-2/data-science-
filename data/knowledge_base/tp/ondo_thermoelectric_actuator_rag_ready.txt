
DOCUMENT: Сервопривод термоэлектрический ONDO

PRODUCT: Термоэлектрический сервопривод
BRAND: ONDO

# Назначение

Термоэлектрический сервопривод ONDO применяется в системах:

- отопления
- вентиляции
- кондиционирования

Устройство используется для изменения положения клапанов и автоматического регулирования температуры в помещениях.

Сервопривод может устанавливаться на:

- отопительные приборы
- гребенки теплого пола
- коллекторы отопления

Питание сервопривода: 230 В.

# Модели

OSPNO220 — нормально открытый (NO)

OSPNC220 — нормально закрытый (NC)

# Технические характеристики

Тип устройства:

NO — нормально открытый
NC — нормально закрытый

Пусковой ток: 0.2 А

Потребляемая мощность: до 3 Вт

Сила нажатия штока: 100 ± 10 %

Время срабатывания: примерно 3.5 минуты

Напряжение питания: 230 В ±10 %

Диапазон рабочей температуры: −10 … +60 °C

Подключение к клапану: резьба M30 × 1.5

Материал корпуса: PC пластик

Рабочая среда: воздух без агрессивных газов и пыли

# Габариты

Диаметр корпуса: около 42 мм

Высота корпуса: около 64 мм

# Конструкция

Основные элементы устройства:

1. шток
2. пружина
3. клапан
4. гайка
5. контакты
6. кабель

# Комплект поставки

В комплект поставки входят:

- сервопривод термоэлектрический — 1 шт
- руководство по эксплуатации — 1 шт
- упаковка — 1 шт

# Монтаж

Место установки должно обеспечивать свободное размещение кабеля подключения.

Кабель должен быть расположен таким образом, чтобы исключить воздействие конденсата.

Установка выполняется в следующей последовательности:

1. Установить сервопривод на вентиль или коллектор.
2. Закрепить устройство вручную с помощью накидной гайки.
3. Подключить устройство к электрической сети.

Монтаж выполняется только при отключенном электропитании.

# Требования безопасности

Работы по монтажу и электрическому подключению должны выполняться квалифицированными специалистами.

Все операции выполняются только при отсутствии напряжения.

Необходимо соблюдать требования электробезопасности.

# Принцип работы

Сервопривод содержит нагревательный элемент.

После подачи напряжения нагревательный элемент нагревает рабочий элемент привода.

После небольшой задержки начинается движение штока.

Для модели NC (нормально закрытый):

- при подаче напряжения клапан открывается
- при отключении питания клапан закрывается

Для модели NO (нормально открытый):

- при подаче напряжения клапан закрывается
- при отключении питания клапан открывается

# Условия хранения и транспортировки

Хранение выполняется согласно условиям хранения 3 ГОСТ 15150.

Транспортировка выполняется согласно условиям 5 ГОСТ 15150.

# Утилизация

Утилизация изделия выполняется в соответствии с действующим законодательством РФ.

# Гарантия

Гарантийный срок: 7 лет (84 месяца) со дня продажи.

Гарантия распространяется на дефекты, возникшие по вине производителя.

Гарантия не распространяется на повреждения, вызванные:

- нарушением правил монтажа
- неправильной эксплуатацией
- механическими повреждениями
- воздействием агрессивных веществ
- посторонним вмешательством в конструкцию изделия
