
DOCUMENT: Краны шаровые усиленного типа для воды СТМ ПРОФИ никелированные

BRAND: СТМ ПРОФИ
PRODUCT: Шаровые краны усиленного типа для воды, никелированные

# Назначение

Шаровые краны СТМ ПРОФИ применяются в качестве запорной арматуры на трубопроводах:

- питьевого водоснабжения
- хозяйственно-питьевого водоснабжения
- горячего водоснабжения
- систем отопления
- систем сжатого воздуха
- трубопроводов жидких углеводородов
- технологических трубопроводов для неагрессивных жидкостей

# Материалы

Корпус крана:
латунь горячепрессованная CW617N с никелированным покрытием

Запорный шар:
латунь хромированная CW617N + Cr

Седельные уплотнения:
Teflon (PTFE)

Сальниковая прокладка:
PTFE + C + EM

Ручка-бабочка:
алюминий

Ручка-рычаг:
сталь с покрытием Dacromet и ПВХ оболочкой

# Рабочие характеристики

Рабочая температура:
-20°C ... +150°C

Класс герметичности затвора:
A (по ГОСТ 9544-2005)

Нормативный срок службы:
25 лет

Минимальный ресурс:
20000 циклов

Наработка на отказ:
45000 циклов

Диапазон условных проходов:
DN 15 – DN 50

Номинальное давление:
PN 2.0 – 4.0 МПа

# Основные типы кранов

Кран шаровой никелированный:

- с внутренней резьбой, ручка-рычаг
- с внутренней резьбой, ручка-бабочка
- с внутренней и наружной резьбой, ручка-рычаг
- с внутренней и наружной резьбой, ручка-бабочка
- с накидной гайкой, ручка-бабочка
- угловой с накидной гайкой, ручка-бабочка

# Монтаж

Кран может устанавливаться в любом положении.

Не допускаются нагрузки от трубопровода:

- изгиб
- сжатие
- растяжение
- кручение
- вибрация
- перекос

Несоосность трубопроводов:

3 мм на длине до 1 м + 1 мм на каждый следующий метр.

Для резьбовых соединений рекомендуется использовать:

- ФУМ-ленту
- льняную прядь

# Эксплуатация

Кран должен эксплуатироваться при температуре и давлении, указанных в технических характеристиках.

Запрещается эксплуатация с ослабленной гайкой крепления рукоятки.

# Хранение

Хранение выполняется по условиям 3 ГОСТ 15150.

# Транспортировка

Транспортировка выполняется по условиям 5 ГОСТ 15150.

# Гарантия

Гарантийный срок:
60 месяцев.
