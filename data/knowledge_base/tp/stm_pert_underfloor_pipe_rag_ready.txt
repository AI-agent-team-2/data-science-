
DOCUMENT: Труба для теплого пола PE-RT СТМ ПЛАСТ

BRAND: СТМ ПЛАСТ
PRODUCT: Труба из полиэтилена повышенной термостойкости (PE-RT)

# Назначение

Труба PE‑RT применяется в системах:

- питьевого водоснабжения
- хозяйственно‑питьевого водоснабжения
- горячего водоснабжения
- водяного низкотемпературного отопления
- систем теплого пола
- теплых стен
- почвенного подогрева
- технологических трубопроводов для жидкостей, неагрессивных к материалу трубы

Трубы PE‑RT обладают высокой гибкостью, устойчивостью к температуре и внутреннему давлению, не подвержены коррозии.

Материал PE‑RT является этилен‑октеновым сополимером с улучшенной молекулярной структурой, обеспечивающей высокую устойчивость к гидростатическим нагрузкам и температуре.

# Монтаж

Монтаж труб должен выполняться при температуре окружающей среды не ниже 5°C.

Для соединения рекомендуется использовать:

- цанговые фитинги
- компрессионные фитинги для металлопластиковых труб

Во время монтажа не допускаются:

- сплющивания трубы
- переломы трубопровода

При повреждении трубы участок должен быть удален.

Бухты труб, хранившиеся при температуре ниже 0°C, перед монтажом необходимо выдержать 24 часа при температуре не ниже 10°C.

Свободные концы труб должны закрываться заглушками для предотвращения попадания грязи.

При изгибе трубы близком к предельному радиусу рекомендуется предварительно прогреть трубу строительным феном до 120°C.

В местах поворота трубы рекомендуется закреплять трубопровод хомутами с шагом 15 см.

Система теплого пола должна заливаться бетонным раствором только после гидравлических испытаний.

Труба во время заливки должна находиться под давлением 0.3 МПа.

Минимальная толщина бетонного слоя над трубой:

3 см.

# Ограничения эксплуатации

Трубы PE‑RT нельзя применять:

- при температуре транспортируемой жидкости выше 80°C
- при давлении выше указанного в технических характеристиках
- в помещениях категории «Г» по пожарной безопасности
- в помещениях с тепловым излучением выше 120°C
- в системах центрального отопления с элеваторными узлами
- для расширительных, предохранительных и сигнальных трубопроводов

# Хранение

Трубы можно перевозить любым видом транспорта.

Бухты труб должны перевозиться в крытом транспорте.

Трубы должны укладываться на ровную поверхность без острых предметов.

Высота штабеля при хранении:

не более 3 м.

Хранение должно выполняться по условиям 5 ГОСТ 15150.

Трубы должны быть защищены от прямых солнечных лучей.
