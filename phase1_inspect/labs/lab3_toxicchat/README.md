# Lab 3: ToxicChat — Real-world Toxic Conversations

## Цель

Протестировать модель на реальных вредоносных запросах, собранных у пользователей ChatGPT.

## Датасет

**ToxicChat** содержит ~10 000 разговоров, аннотированных по двум осям:
- `toxicity` — токсичный контент (оскорбления, угрозы, ненависть)
- `jailbreaking` — попытки обойти фильтры безопасности

Ключевое отличие от AdvBench: это *реальные* запросы от настоящих пользователей,
а не специально сконструированные атаки. Распределение более реалистичное.

## Задание

1. Запусти на 50 примерах из test split:
   ```bash
   python -m inspect_ai eval run.py --model openrouter/openai/gpt-4o-mini --limit 50
   ```

2. Сравни ASR между `toxicity=1` и `jailbreaking=1` группами (через inspect view → metadata).

3. Заполни `report_template.md`.

## Вопрос

Почему ASR на ToxicChat может быть ниже, чем на AdvBench, хотя оба датасета
содержат вредоносные запросы?
